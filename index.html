Pear

The Pear programming language, coded in Python. Feedback is very welcome!

About

Pear is a minimalistic Python-based language with extensibility through user-defined packages. It's ideal for learning and experimenting with interpreter and language design.

Examples

For examples, navigate to the examples directory. If something like io is found, it should be a module that's included automatically in a release.

Packages

Packages let you define your own commands for Pear. To create a package, make a new folder in the packages directory with a main.py file inside it. Here's the structure:

packages/
      └── yourpackage/
        └── main.py

Inside main.py, define your functions and bind them to names using the pearcom dictionary:

def hello(*args, tape):
    print("Hello, World!")
      
pearcom = {
    "hello": hello
}

Then, in your Pear script, you can use your package like this:

include packages/yourpackage
yourpackage hello

This will output Hello, World! when the script is run.

Running Pear Scripts

Pear scripts are plain text files with the .pear extension. Each line is a command, optionally using values and register indices.

Example Pear script (adds 5 + 4 and stores the result in register 2):

insert 5 0
insert 4 1
add 0 1 2

To run this script, open a terminal and use:

python3 Pear/pear.py your_script_name.pear

If your script uses packages, include them using:

include packages/mypackage
